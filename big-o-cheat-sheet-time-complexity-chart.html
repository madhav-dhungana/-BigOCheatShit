How to know which algorithm has which time complexity? Here is the cheatsheet for time complexity calculationðŸš€

  • When the calculation is not dependent with the input size, it will be a constant time complexity (O(1)).
  • When the input size is reduced by half, when iteration, handling recursion, etc..., it will be a logarithmic time complexity (O(log n)).
  • When there is a single loop within a algorithm, it will be linear time complexity (O(n)).
  • When there are nested loops within the algorithm, which means a loop in a loop, it will be quadratic time complexity (O(n^2))..
  • When the rate of growth doubles with each addition to the input, it will be exponential time complexity (O2^n).
